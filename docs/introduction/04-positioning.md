# Positioning

## What Category Does Drumee Belong To?

Drumee creates its own category: **Sovereign Data Infrastructure**.

| What people compare it to | Why that's incomplete |
| :---- | :---- |
| Google Drive / Dropbox | File storage tools. Drumee is full infrastructure with collaboration and extensibility built in. |
| Notion / Confluence | Document SaaS. Drumee is self-hosted infrastructure — not another app. |
| Slack | Communication tool. Drumee's chat is folder-native — tied to the data context it belongs to. |
| Nextcloud | Closest technically — but file sync + apps bolted on. Drumee is architecture-first with a unified OS model. |

## The One Mental Model You Need

> A Drumee folder is not just a folder. It is a workspace: it has files, a built-in chat channel, a permission system, and an activity log — all inside one unit. Every workspace  is a sovereign, self-contained collaboration space.


This changes how you think about work:

- Not: *"Where do I store this file?"*

- But: *"What workspace does this work belong to?"*

The folder becomes the organizing unit for people, files, conversations, and permissions — all in one place, all on infrastructure you own.
